map2(func, *args)

 

Apply a function to arguments, like built-in map and return a tuple with the results.

Parameters:
  • func - Function to apply (callable).
  • args - Arguments to apply (list, tuple, ...).
Returns:
Function results (tuple).

Note: Python 3+ map returns a map object, an iterator-like object which generates the map result only once. Converting the map object into a tuple restores the Python 2 behavior.

      • geodesy.bases.LatLonHeightBase: (INTERNAL) Base class for LatLon points on spherical or ellipsiodal earth models.

          • geodesy.ellipsoidalVincenty.LatLon: Using the formulae devised by Thaddeus Vincenty (1975) with an ellipsoidal model of the earth to compute the geodesic distance and bearings between two given points or the destination point given an start point and initial bearing.

          • geodesy.ellipsoidalNvector.LatLon: An n-vector-based ellipsoidal LatLon.
          • geodesy.sphericalNvector.LatLon: New n-vector based point on spherical earth model.
        • geodesy.sphericalBase.LatLonSphericalBase: (INTERNAL) Base class for spherical Latlons.
          • geodesy.sphericalNvector.LatLon: New n-vector based point on spherical earth model.
          • geodesy.sphericalTrigonometry.LatLon: New point on spherical model earth model.
      • geodesy.mgrs.Mgrs: Military Grid Reference System (MGRS/NATO) references, with method to convert to UTM coordinates.
      • geodesy.osgr.Osgr: OSGR coordinate.
      • geodesy.utm.Utm: Universal Transverse Mercator (UTM) coordinate.
      • geodesy.vector3d.Vector3d: Generic 3-D vector manipulation.
        • geodesy.ellipsoidalBase.CartesianBase: (INTERNAL) Base class for ellipsoidal Cartesians.
          • geodesy.ellipsoidalNvector.Cartesian: Extended to convert geocentric Cartesian points to to Nvector and n-vector-based ellipsoidal LatLon.
          • geodesy.ellipsoidalVincenty.Cartesian: Extended to convert geocentric Cartesian points to Vincenty-based ellipsoidal LatLon.
        • geodesy.nvector.Nvector: Base class for ellipsoidal and spherical Nvector.
          • geodesy.ellipsoidalNvector.Nvector: An n-vector is a position representation using a (unit) vector normal to the earth ellipsoid.
          • geodesy.sphericalNvector.Nvector: An n-vector is a position representation using a (unit) vector normal to the earth's surface.

        • geodesy.datum.Datum: Datum ellipsoid and transform.
        • geodesy.datum.Ellipsoid: Ellipsoid with semi-major, semi-minor axis inverse flattening.
        • geodesy.lcc.Lcc: Lambert conformal conic East-/Northing location.
        • geodesy.datum.Transform: Helmert transformation.
    • exceptions.BaseException: Common base class for all exceptions

        • geodesy.lcc.Lcc: Lambert conformal conic East-/Northing location.
